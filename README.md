#TripTopia
Trip Topia is a modern travel website built to help users explore destinations, 
discover travel experiences, and plan their trips easily. The project features a clean and responsive user interface
designed to provide an engaging travel planning experience.

##Features##
-Responive design
-Travel destinations
-Easy navigation
-Attractive travel-themed UI

##Technologies Used##
-HTML
-CSS
-JavaScript

##Project Goal##
The goal of TripTopia is to provide travelers with an easy to use platform
for discovering destinations and planning memorable journeys.
